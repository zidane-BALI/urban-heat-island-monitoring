# urban-heat-island-monitoring
This project presents an autonomous mobile meteorological sensor network developed for a laboratory at the IUT of Mulhouse. The system is designed to collect high-resolution environmental data across the city to support the study of Urban Heat Islands (UHI).

The sensor node is powered by a solar panel and battery, enabling long-term autonomous operation. It measures key meteorological parameters and transmits the collected data wirelessly using LoRa for long-range communication and Wi-Fi when available.

The project combines embedded systems, wireless communication, renewable energy, and environmental monitoring to provide researchers with reliable field data for urban climate analysis.
